# ClassFlow（武汉商学院专版）

ClassFlow 是基于武汉商学院教务系统深度定制的课程表应用。
本仓库仅保留武汉商学院（WBU）相关能力，不再维护多学校适配逻辑。

## 当前定位
- 仅支持武汉商学院教务自动化对接
- 保留课程表、今日课程、调课、课程管理、提醒、小组件等核心功能
- 保留桌面小组件能力（可直接添加到系统桌面）

## 更新策略
- 应用内提供“手动检查版本更新”入口
- 用户可自行选择是否下载新版本
- 新版本通过 GitHub Releases 分发

## 仓库说明
- 已移除与多学校适配相关的冗余文档与无关内容
- README 保持最小化，只描述当前专版范围
